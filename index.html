About Fsoc-Medium

Your comprehensive task management and productivity platform

Fsoc-Medium is a modern, responsive web application designed to help you manage your daily tasks efficiently while staying informed about weather conditions. Built with vanilla JavaScript, HTML5, and CSS3, it provides a clean, intuitive interface for productivity.

Features

Task Management

Create, edit, and organize your tasks with our intuitive task manager. Mark tasks as complete and filter by status.

Weather Widget

Stay informed about weather conditions in any city with our integrated weather widget powered by OpenWeatherMap API.

Dark Mode

Switch between light and dark themes for comfortable viewing in any environment.

Responsive Design

Fully responsive design that works perfectly on desktop, tablet, and mobile devices.

Inline Editing

Edit your tasks directly in place with our inline editing functionality and keyboard shortcuts.

Modern UI

Beautiful, modern interface with smooth animations and intuitive user experience.
